# 第五關「卦名填空」實作計畫

## 目標 (Goal)
新增第五個學習測驗「卦名填空」，讓使用者練習將卦的屬性（如天地風火山水雷澤）與卦名連結。

## 設計邏輯 (Design Logic)
- **題型**：題目會根據卦名長度（多數3字如「水雷屯」，少數純卦如「乾為天」），挖空前 1 個或 2 個代表「上下卦屬性」的字。
- **作答區**：上方顯示帶有空格（Slot）的題目，例如 `[水] [ ] [屯]` 或 `[ ] [ ] [屯]`。純卦如 `乾為天` 則挖空 `天` 變為 `乾為 [ ]`。
- **選項區**：與「畫卦天才(Game 3)」共用或使用相同的 8 個按鈕選項。選項內容可設定為「卦象 (☰☱...)」或「卦屬性 (天地...)」。選項位置可設定「固定」或「隨機」。

## 實作步驟 (Implementation Steps)

### `index.html`
1. **主選單**：新增 `<button class="game-mode-btn" data-game="5">卦名填空</button>`。
2. **設定區塊**：新增 `<div id="settings-g5" class="game-settings" style="display:none;">`
   - 挖空數量：挖空一字 / 挖空兩字 / 隨機混合
   - 選項呈現：顯示卦象 / 顯示屬性
   - 選項位置：固定 / 隨機
3. **測驗區塊**：在 `#quiz-active` 中新增針對 Game 5 的版面，或是復用 Game 3 的插槽 `slot` 概念，但排版改為橫向（因為是文字填空）。

### `app.js`
1. **`quizConfig` 更新**：加入 `g5Blank`, `g5OptShow`, `g5Pos`。
2. **Setup 選單切換**：點擊跳至 Game 5 選單。
3. **`generateQuiz(5)`**：
   - 每題根據 `g5Blank` 決定要建立幾個填空。
   - 純卦（乾為天）只有一個屬性字，如果抽到 `挖空兩字`，應該強制只挖空一字或重新抽題，為簡化，純卦永遠只挖一字。
4. **`loadQuestion()` 和 作答邏輯**：
   - 產生橫向排列的 Question 插槽。
   - 產生下方 8 個按鈕選項。
   - 點擊按鈕填入目前 active 的插槽。
   - 新增 `btn-submit-fill` 用來提交並批改。
